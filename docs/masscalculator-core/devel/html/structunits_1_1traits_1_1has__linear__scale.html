template<typename... T>
struct units::traits::has_linear_scale< T >

Trait which tests whether a type is inherited from a linear scale.

Inherits from std::true_type or std::false_type. Use has_linear_scale<U1 [, U2, ...]>::value to test one or more types to see if they represent unit_t's whose scale is linear.

Template Parameters
Tone or more types to test.
